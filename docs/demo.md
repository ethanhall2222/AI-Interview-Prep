# 2-Minute Demo Script

## 0:00-0:20 Landing and flow framing
1. Open `/`.
2. Point out the workflow rail and the product path: Job Lab -> Resume Review -> Practice -> Tracker.

Talk track:
- "This flow turns a job post into resume fixes, then interview prep, then pipeline tracking."

## 0:20-0:55 Job Lab reliability
1. Open `/jobs`.
2. Mention timeout + retry behavior and manual fallback if generation fails.

Talk track:
- "Even if AI generation fails, the UI gives a clear retry path and keeps the workflow moving."

## 0:55-1:30 Resume Review premium outputs
1. Open `/resume-review`.
2. Click `Use sample resume`, `Use sample job description`, then `Run resume review`.
3. Show:
- "Fix this first" priority panel
- Coverage and bullet quality scores
- Missing/present/overused keyword panels
- Bullet quality checks and before/after rewrites
- `Copy full review` and `Export txt`

Talk track:
- "Outputs are explainable and deterministic, which is useful for interview demos and regression testing."

## 1:30-1:50 Practice reliability
1. Open `/practice`.
2. Show voice/type toggle and permission status.

Talk track:
- "If browser voice features are unavailable, users automatically continue in Type mode instead of getting blocked."

## 1:50-2:00 Tracker close
1. Open `/tracker`.
2. Add a sample entry and change its status.

Talk track:
- "This closes the loop with a lightweight, fast tracker for application momentum."

## Admin Extension
1. Open `/admin/agents` as an admin.
2. Show the continuous build lanes: Discover, Shape, Build, Verify, Launch, Learn.
3. Use the reusable build prompt to start the next product iteration.

Talk track:
- "The product now has an agency-style operating model so future work has clear roles, handoffs, and release gates."
